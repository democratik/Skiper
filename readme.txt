HOW TO INSTALL THE EXTENSION / ЯК ВСТАНОВИТИ РОЗШИРЕННЯ
==================================================

[ ENGLISH ]

How to install the Skipper extension in Chromium-based browsers (Chrome, Edge, Brave):
1. Open your browser.
2. Go to the extensions management page:
   - For Chrome: type `chrome://extensions/` in the address bar and press Enter.
   - For Edge: type `edge://extensions/` in the address bar and press Enter.
3. Enable "Developer mode" (usually a toggle switch located in the top right corner of the page).
4. Click the "Load unpacked" button that appears at the top left.
5. Select the folder containing all the extension files and click "Select Folder".
* Tip: Click the puzzle piece icon in the top right corner of your browser and pin the extension to easily access the settings popup.

How to install the extension in Mozilla Firefox:
1. Open Mozilla Firefox.
2. Type `about:debugging` in the address bar and press Enter.
3. Click on "This Firefox" in the left sidebar.
4. Click the "Load Temporary Add-on..." button.
5. Open your extension folder, select ANY file inside it (e.g., `manifest.json`), and click "Open".
* Note: Firefox installs unpacked extensions temporarily. You will need to reload it using these steps after restarting the browser.

[ УКРАЇНСЬКА ]

Як встановити розширення Skipper у браузерах на базі Chromium (Chrome, Edge, Brave):
1. Відкрийте ваш браузер.
2. Перейдіть на сторінку керування розширеннями:
   - Для Chrome: введіть `chrome://extensions/` в адресний рядок і натисніть Enter.
   - Для Edge: введіть `edge://extensions/` в адресний рядок і натисніть Enter.
3. Увімкніть "Режим розробника" / "Developer mode" (перемикач у верхньому правому куті сторінки).
4. Натисніть кнопку "Завантажити розпаковане" / "Load unpacked" (у верхній лівій частині екрана).
5. Виберіть папку, в якій знаходяться всі файли розширення, і натисніть "Вибрати папку".
* Порада: Натисніть на іконку пазла у верхньому правому куті браузера та закріпіть розширення (Pin), щоб мати швидкий доступ до налаштувань.

Як встановити розширення в Mozilla Firefox:
1. Відкрийте Mozilla Firefox.
2. Введіть `about:debugging` в адресний рядок і натисніть Enter.
3. Натисніть на пункт "Цей Firefox" (This Firefox) у лівому бічному меню.
4. Натисніть кнопку "Завантажити тимчасовий додаток..." (Load Temporary Add-on...).
5. Відкрийте папку з розширенням, виберіть БУДЬ-ЯКИЙ файл всередині (наприклад, `manifest.json`) і натисніть "Відкрити".
* Примітка: Firefox встановлює розпаковані розширення тимчасово. Вам доведеться завантажити його знову після перезапуску браузера.
